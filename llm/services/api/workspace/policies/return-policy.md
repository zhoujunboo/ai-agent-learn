# Return Policy

Customers may request a return within 7 days after receiving eligible goods.
Returned items should be unused, complete, and include original packaging,
accessories, invoice, and gifts. Customized goods, opened hygiene products,
digital goods, and items marked final sale are not eligible for no-reason return.

For quality issues, logistics damage, wrong shipment, or missing accessories,
customer service should verify the order, product photos, and delivery record
before approving a return.
