# After-Sale FAQ

Q: What information is needed for a return review?
A: Order number, product ID, reason, photos if needed, delivery status, and
whether the item is unused with full packaging.

Q: Can a customer return an opened item?
A: It depends on category and reason. Opened hygiene products or customized
goods are usually not eligible unless there is a verified quality issue.

Q: Where should customer service write an analysis result?
A: Write reports or ticket notes under workspace/tickets using a readable file
name such as tickets/{orderId}-analysis.md.
